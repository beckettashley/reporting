# Reporting Dashboard

Reporting prototype for the headless ecommerce CMS platform. Surfaces funnel performance, variant experiment results, and conversion metrics for both internal teams and merchants.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4** + shadcn/ui
- **Recharts** for data visualization
- **Vercel** for hosting
